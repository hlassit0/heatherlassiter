Learning Linux & Virtual Machines

This project involved setting up and configuring a virtual machine (using VirtualBox) with an Ubuntu Linux environment. I focused on understanding fundamental Linux commands, navigating the file system, managing processes, and installing basic security tools. This foundational work is crucial for any cybersecurity enthusiast, providing a sandboxed environment to experiment without affecting the host system.

Key learning points included:

  • Virtualization concepts and benefits.
  • Basic Linux shell commands (ls, cd, mkdir, rm, mv, cp).
  • User and permission management.
  • Package management (apt-get).
  • Introduction to network configuration within a VM.

EEG Data Analysis

This project involved processing and analyzing EEG (Electroencephalography) data using the MNE-Python library. I focused on common signal processing steps such as filtering raw EEG data to remove artifacts, epoching continuous data into event-related segments, and performing power spectral density analysis to identify and quantify neural oscillations, particularly in the alpha band.

This work provided hands-on experience with:

  • Biosignal processing fundamentals.
  • Using the MNE-Python library for neuroimaging data.
  • Time-frequency analysis and interpretation of EEG features.
  • Applying scientific programming techniques in Python.

Music Database Management

In this project, I designed and implemented a relational database schema using MySQL for managing music-related data, including artists, albums, and songs. The project involved creating tables, defining relationships (e.g., one-to-many between artists and albums, albums and songs), and populating the database with sample data.

A key part of this project was developing SQL queries to retrieve specific information, such as:

  • Retrieving songs by a specific artist.
  • Listing albums released in a particular year.
  • Creating views for aggregated data, like "Top 100 Songs" based on a `chart_position` field.
